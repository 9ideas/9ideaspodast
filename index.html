About The Podcast

Since the start of 2018, Seth & Ken host an informal discussion of all things related to application security. So far, it seems to be mostly opinions, biases, and recommendations about the security industry, current events, and anything else we come up with. Guests include industry professionals ranging from consultants to managers.
